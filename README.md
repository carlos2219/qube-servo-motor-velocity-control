# QUBE Servo Motor Velocity Control

This repository documents a motor velocity control practice developed using the Quanser QUBE-Servo 3 platform during the Minor in Autonomy of Unmanned Aerial Vehicles at Tecnológico de Monterrey.

The practice focuses on the design, analysis, and implementation of an Integral-plus-Proportional (I+P) controller for motor velocity tracking. It includes the derivation of the control law, closed-loop transfer function analysis, controller gain calculation from transient response specifications, MATLAB-based computations, experimental validation, and controller tuning based on real system behavior.

The work also includes a comparison between I+P and classical PI control strategies, highlighting the practical advantage of the I+P formulation in reducing the setpoint kick effect and limiting excessive control action. The controller was implemented and tested in Simulink and QUARC using the Quanser QUBE-Servo 3 platform.

This repository is intended to present both the theoretical development and practical implementation of real-time motor velocity control on an experimental mechatronic system.

## Topics Covered
- Motor velocity control
- Integral-plus-Proportional (I+P) control
- Closed-loop transfer function analysis
- Controller gain design from performance specifications
- MATLAB calculations
- Simulink and QUARC implementation
- Experimental tuning
- I+P vs PI controller comparison
